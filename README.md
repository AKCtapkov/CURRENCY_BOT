# Currency Converter Bot

Telegram-бот для конвертации валют (доллар, евро, рубль) по актуальному курсу.

## Команды
- `/start`, `/help` – инструкция по использованию
- `/values` – список доступных валют

## Как использовать
Отправьте боту сообщение в формате: <исходная валюта> <целевая валюта> <количество>
